## 📊 시스템 기능

### 1. 기본 진실성 측정
- **5가지 검증 방법**:
  - 사실적 일관성 (30% 가중치)
  - 논리적 일관성 (25% 가중치)
  - 시간적 일관성 (20% 가중치)
  - 의미적 분석 (15% 가중치)
  - 통계적 분석 (10% 가중치)

### 2. 고급 탐지 시스템
- **메타-진실성 탐지기**: 진실성 탐지기 자체의 거짓말 탐지
- **종교적 맥락 인식**: 종교적/신학적 주제에 대한 적절한 처리
- **강화된 과학적 사실 검증**: 기본 과학 사실에 대한 정확한 판단
- **의도적 거짓말 탐지**: 질문자의 의도적 거짓말 시도 감지
- **인간 행동 패턴 인식**: 인간의 기본 능력에 대한 명확한 판단
- **선의의 거짓말 인식**: 거짓말의 복잡성과 선의적 측면 이해
- **맥락 인식 탐지**: 단어의 모순성을 인식하는 시스템
- **복합 문장 분석**: 여러 맥락이 섞인 문장의 분리 분석
- **코딩 품질 탐지**: 프로그래밍 코드의 진실성 검증
- **다국어 분석**: 여러 언어로 된 문장의 진실성 탐지
- **말장난 탐지**: 언어적 유머와 말장난의 진실성 분석

### 3. 자동 교정
- 1% 이상 거짓말 감지 시 자동 교정
- 과장된 표현 완화
- 논리적 모순 제거
- 시간 표현 명확화

### 4. 시각화
- 진실성 트렌드 차트
- 검증 방법별 점수
- 상세 분석 결과 테이블

## 🔍 분석 예시

### 진실한 문장
```
입력: "지구는 둥글다."
결과: 72.0% 진실성, 교정 불필요
```

### 거짓 문장
```
입력: "지구는 평평하다."
결과: 51.0% 진실성, 교정 필요
교정문: "지구는 대부분 평평하다."
```

### 과장된 표현
```
입력: "정말로 완전히 절대적으로 모든 것이 100% 진실이다."
결과: 55.0% 진실성, 교정 필요
교정문: "상당히 대부분의 것이 거의 진실이다."
```
